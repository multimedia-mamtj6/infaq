# Images Folder

Place your project images in this folder with the following filenames:

## Plan Drawings (Lukisan Pelan)
- `plan-front-view.jpg` - Pelan Pandangan Hadapan
- `plan-side-view.jpg` - Pelan Pandangan Kanan & Kiri
- `plan-floor.jpg` - Pelan Lantai
- `plan-roof.jpg` - Pelan Bumbung

## Project Location (Lokasi Projek)
- `location-main-entrance.jpg` - Tapak dari pintu masuk utama
- `location-satellite.jpg` - Imej satelit
- `location-site-plan.jpg` - Pelan tapak projek

## Image Requirements
- Supported formats: JPG, JPEG, PNG, WebP
- Recommended aspect ratio:
  - Plan drawings: 16:9 (landscape)
  - Location images: 1:1 (square)
- File size: Keep images optimized for web (under 500KB each recommended)

## Alternative: Using External URLs
If you prefer to host images externally, you can edit `infaq-pembangunan.html` and replace the `src` attributes with full URLs:
```html
<img src="https://your-domain.com/images/plan-front-view.jpg" ...>
```
